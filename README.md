# Warframe Calculator — PWA

A mobile-first DPS + Probability calculator for Warframe.

## Files
- `index.html` — the app
- `manifest.json` — makes it installable as a PWA on Android

## How to host for free (GitHub Pages)

1. Create a free account at https://github.com
2. Click **New repository**, name it `warframe-calc`, make it **Public**
3. Upload `index.html` and `manifest.json`
4. Go to **Settings → Pages → Source → main branch → Save**
5. Your app will be live at: `https://YOUR_USERNAME.github.io/warframe-calc/`

## How your friend installs it on Android

1. Open the link above in **Chrome** on Android
2. Tap the **⋮ menu** (top right)
3. Tap **"Add to Home Screen"**
4. Done — it installs like a real app with its own icon, no browser bar

## Alternatively: Netlify (even easier)

1. Go to https://netlify.com and sign up free
2. Drag and drop the `warframe-calculator` folder onto the dashboard
3. Netlify gives you a live URL instantly

## Features
- **DPS Calculator** — Damage, Multishot, Fire Rate, Crit Chance/Damage, Status Chance
- **Probability Calculator** — Solve for drop rate, attempt count, or projected success rate
- **Weapon Presets** — Hull Buster cannon/shotgun variants pre-loaded
- **Guide tab** — explains what each stat means
- Works offline after first load
